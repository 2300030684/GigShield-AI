# 🚀 TrustPay

### AI-Powered Parametric Insurance for Gig Workers

---

## 📌 About the Project

TrustPay is an AI-driven parametric insurance platform designed to protect gig workers (like Zomato/Swiggy delivery partners) from unpredictable income loss caused by weather and environmental disruptions.

---

## 💡 Inspiration

Gig workers depend on daily earnings, but factors like heavy rain, extreme heat, and air pollution can reduce their income by 20–30%.
We built TrustPay to provide **instant financial protection using AI and automation**.

---

## 🧠 What We Built

* 🔍 AI-based risk prediction
* 💰 Dynamic premium pricing
* 🌦️ Real-time environment monitoring
* ⚡ Automatic claim triggering
* 💳 Instant payout simulation
* 🛡️ Fraud detection system

## ⚙️ How It Works

1. User registers on the platform
2. AI calculates risk score
3. Weekly premium is assigned
4. System monitors real-time conditions
5. Disruption detected → claim triggered
6. Income loss calculated
7. Instant payout processed

---

## 📊 Parametric Triggers

| Condition       | Threshold |
| --------------- | --------- |
| 🌧️ Rainfall    | > 50 mm   |
| 🌡️ Temperature | > 45°C    |
| 🌫️ AQI         | > 300     |

## 🚀 How to Run

### 🔹 Backend (Spring Boot)

1. Open in Eclipse
2. Run:

```
BackendApplication.java → Run As → Java Application
```

Runs at:

```
http://localhost:8080
```

---

### 🔹 Frontend

#### HTML Version

* Open with Live Server

#### React Version

```
cd frontend
npm install
npm start
```

Runs at:

```
http://localhost:3000
```
